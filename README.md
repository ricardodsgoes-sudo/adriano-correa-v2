# Adriano Ademicon — Site profissional

Site corporativo para consultoria de consórcios. Landing page educacional e de captação de leads para Adriano.

## 🎨 Design

- **Tema:** Light mode corporativo
- **Cores:** Vermelho vibrante (#FF0000), branco, preto
- **Tipografia:** Sora (body) + Hanken Grotesk (display)
- **Estilo:** Direto, limpo, sem fluff
- **Mobile-first:** Responsivo em todos os devices

## 📁 Estrutura

```
src/
├── app/
│   ├── layout.tsx       — Layout principal
│   ├── page.tsx         — Página home
│   └── globals.css      — Estilos globais
└── components/
    ├── Header.tsx       — Navegação fixa
    ├── Hero.tsx         — Seção inicial
    ├── ConsorcioVsCredito.tsx  — Educação
    ├── Servicos.tsx     — Serviços
    ├── ComoFunciona.tsx — Processo
    ├── CTA.tsx          — Call-to-action
    └── Footer.tsx       — Rodapé
```

## 🚀 Como rodar

```bash
npm install
npm run dev
# Abrir http://localhost:3000
```
